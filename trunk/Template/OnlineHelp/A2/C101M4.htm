1.1      案件設定

1.1.1     新增案件申辦項目

         操作步驟：

1.  

在後台系統管理畫面，左側欄的申辦案件管理子選單，點選「案件設定」功能。

                圖表「後台管理選單」畫面

 

2.  

在案件設定畫面，點選「新增」鈕，建立醫事人員與醫事機構的申辦項目資訊與案件相關設定。

                 圖表「案件設定-新增」畫面

 

3.  

進入新增畫面，輸入【服務同意需知文字】及選擇【分類】、【申辦項目辦理縣市】、【申辦項目名稱】、【是否需服務同意說明】、【是否可線上申辦】、【是否可表單下載】、【是否需繳費規費】、【是否需領取】、【點收權限對應群組】、【是否需要分文】、【分文權限對應群組】、【案件處理權限對應群組】，點選「儲存」鈕，系統提示案件新增成功，即完成案件設定新增。

                    圖表「案件設定-新增內頁」畫面

 

4.  

在案件設定畫面，可檢視完成新增結果顯示於下方列表。

                      圖表「案件設定-完成新增結果」畫面

 

1.1.2     查詢及編修案件申辦項目

1.   

在案件設定畫面，輸入複合查詢條件【申辦縣市】、【申辦種類】、【申辦項目】，點選「送出查詢」鈕，進行查詢，將查詢結果顯示於下方列表。

                  圖表「案件設定-查詢結果」畫面

 

2.   

在案件設定畫面，選擇單筆資料，點選「執業」連結，進入申辦項目設定畫面。

                     圖表「案件設定-申辦項目設定」畫面

 

3.    在編輯畫面，可修改【是否需服務同意文字】或選擇【是否需服務同意說明】、【是否可線上申辦】、【是否可表單下載】、【是否需繳費規費】、【是否需領取】、【案件最高權限對應群組】、【是否需要點收】、【點收權限對應群組】、【是否需要分文】、【分文權限對應群組】、【案件處理權限對應

群組】，點選「儲存」鈕，系統提示案件更新成功，即完成資料修改。

    圖表「案件設定-申辦項目設定」畫面

 

1.2      案件設定(申辦項目說明)

1.2.1     新增案件申辦項目說明內容

        操作步驟：

1.    在案件設定畫面，點選「項目說明」鈕，進入項目說明畫面。

 

 

 

 

 

 

 

 

 


                 圖表「案件設定-申辦項目說明」畫面

 

2.   

在項目說明畫面，點選「新增」鈕，建立案件申辦說明內容。

              圖表「案件設定-申辦項目說明-新增」畫面

 

3.    進入新增畫面，輸入【說明內容】及選擇【項目類別】、【說明標題】，點選「儲存」鈕，系統提示案件項目說明新增成功，即完成案件說明內容新增。

 

 


                           圖表「案件設定-申辦項目說明-新增內頁」畫面

 

4.   

至前台申辦畫面，點選「申辦方式說明」功能，選擇【申辦種類】、【申辦項目】，再點選「＋」鈕，可展開檢視已完成新增後的結果。

                 圖表「申辦方式說明-完成新增結果」畫面

 

1.2.2     編修案件項目說明內容

1.   

在申辦項目說明畫面，選擇單筆資料，點選「編輯」鈕，進入編輯畫面。

                  圖表「案件設定-申辦項目說明-編輯」畫面

 

2.   

在編輯畫面，可修改【說明內容】，點選「儲存」鈕，系統提示案件項目說明更新成功，即完成資料修改。

                            圖表「案件設定-申辦項目說明-編輯內頁」畫面

 

3.    至前台申辦畫面，點選「申辦方式說明」功能，選擇【申辦種類】、【申辦項目】，再點選「＋」鈕，可展開檢視已完成修改後的結果。


                 圖表「前台-申辦項目說明-完成編輯結果」畫面

 

1.2.3     刪除申辦項目說明內容

1.   

在申辦項目畫面，選擇資料，點選「編輯」鈕，進入「編輯」畫面。

圖表「案件設定-申辦項目說明-刪除」畫面

 

2.   

在編輯畫面，點選「刪除」鈕，系統提示案件項目說明已刪除。即完成資料刪除。

                             圖表「案件設定-申辦項目說明-刪除內頁」畫面
