## 🔍 如何使用 HTML 比較結果

### 打開方式

1. 使用瀏覽器打開 HTML 文件
2. 可以看到彩色編碼的比較結果：
   - 🟢 綠色：完全相同
   - 🟡 黃色：兩資料夾不同 / 文字檔內容不同
   - ⚪ 灰色：僅左側或僅右側存在

### 查找特定文件

1. 使用瀏覽器的搜索功能（Ctrl+F）
2. 搜索文件名或路徑
3. 查看比對結果和日期

### 查看文件差異詳情

1. 點擊文件名的超連結
2. 會打開該文件的詳細比較結果
3. 可以看到具體的差異內容

## 📝 報告生成方法

這些報告是基於以下方法生成的：

1. **比較工具**: WinMerge
2. **比較對象**:
   - trunk: `F:\AITest\MOH\CodeBase\trunk_DISK_trunk\trunk` vs `F:\AITest\MOH\CodeBaseNew\trunk`
   - e-service: `F:\AITest\MOH\CodeBase\109_e-service` vs `F:\AITest\MOH\CodeBaseNew\e-service`
3. **分析方法**: 
   - 使用 WinMerge 生成 HTML 比較報告
   - 逐行分析 HTML 結果
   - 提取關鍵差異信息
   - 生成結構化的 Markdown 報告
