## 🏗️ 技術架構

### 前端技術

- **框架**: ASP.NET MVC 5
- **視圖引擎**: Razor
- **UI框架**: Bootstrap 3.x
- **JavaScript**: jQuery, jQuery Validation

### 後端技術

- **語言**: C# (.NET Framework 4.7.2)
- **資料存取**: Dapper (輕量級 ORM) + ADO.NET
- **資料庫**: Microsoft SQL Server 2016+
- **郵件服務**: System.Net.Mail
- **文件產生**: Spire.Doc, NPOI
- **排程**: Hangfire

### 架構模式

```
使用者 → View (Razor) → Controller → DAO → Database
                           ↓
                      ViewModel
                           ↓
                      FormModel
```
